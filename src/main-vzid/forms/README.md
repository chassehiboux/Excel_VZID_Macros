# Формы новой надстройки

В этой папке должны лежать экспорты только тех `UserForm`, которые реально остаются в новой архитектуре.

На старте сюда не переносится `frmTerritory`, потому что его заменяет dropdown выбора региона на вкладке `VZID`.

Важно:

- если экспортируешь форму из VBA, почти всегда нужно класть в репозиторий сразу оба файла: `.frm` и `.frx`;
- сборщик импортирует форму по `.frm`, а VBA сам подтягивает соседний `.frx`;
- если `.frm` ссылается на `.frx`, а `.frx` забыт, сборка теперь падает с явной ошибкой.

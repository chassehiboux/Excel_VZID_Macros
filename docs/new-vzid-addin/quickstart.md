# Quickstart: новая надстройка VZID

## Назначение документа

Этот документ будет описывать быстрый запуск разработки и выпуска новой надстройки.

Базовое продуктовое согласование уже зафиксировано, ниже описан текущий рабочий каркас.

## Базовый вектор

- Надстройка строится как новый проект.
- Репозиторий один и публичный: код, релизы и установщик лежат здесь же.
- Главный интерфейс переносится в единый вход.
- Обновление планируется через GitHub Releases.
- `MainVZID.xlam` остаётся единственной Excel-надстройкой продукта.
- Отдельный `updater.exe` отвечает только за замену файла после закрытия Excel.
- Сложные сценарии допускают отдельные окна, но не как главный способ навигации.

## Что будет добавлено после согласования

- структура проекта;
- правила локальной сборки;
- правила выпуска релиза;
- инструкция для тестовой установки;
- инструкция для отката;
- инструкция для публикации новой версии.

## Текущая сборка каркаса

```powershell
python -m pip install --user -r requirements-build.txt
python scripts/build_release.py
```

Результат:

- `build/MainVZID.xlam`
- `build/release/manifest.json`
- `build/release/setup.exe`
- служебный `updater.exe` для упаковки в установку

## Текущий минимальный цикл сопровождения

```powershell
python scripts/build_addins.py --output-dir build_manual_check
python scripts/set_release_version.py 0.2.3
python scripts/publish_release.py
```

# Quickstart: новая надстройка VZID

## Назначение документа

Этот документ будет описывать быстрый запуск разработки и выпуска новой надстройки.

Пока продукт не согласован, здесь фиксируется только базовый вектор.

## Базовый вектор

- Надстройка строится как новый проект.
- Главный интерфейс переносится в единый вход.
- Обновление планируется через GitHub Releases.
- Загрузчик должен быть минимальным и безопасным.
- Сложные сценарии допускают отдельные окна, но не как главный способ навигации.

## Что будет добавлено после согласования

- структура проекта;
- правила локальной сборки;
- правила выпуска релиза;
- инструкция для тестовой установки;
- инструкция для отката;
- инструкция для публикации новой версии.
